# AI 작업 지침

이 디렉터리는 Mimir 문서를 다룰 때 사용하는 AI 작업 지침을 모은다.

## 문서 목록

| 문서 | 용도 |
|------|------|
| `doc-writer.md` | 문서 작성 기준 |
| `multi-tenancy-advisor.md` | 멀티 테넌시 설계 기준 |
| `capacity-planner.md` | 용량/성능 검토 기준 |
| `troubleshooter.md` | 문제 해결 절차 |

## 언제 보나

- 새 문서를 쓸 때 `doc-writer.md`
- tenant 분리와 인증을 설계할 때 `multi-tenancy-advisor.md`
- 저장소/용량 계획을 잡을 때 `capacity-planner.md`
- 장애 분석이나 재현이 필요할 때 `troubleshooter.md`
