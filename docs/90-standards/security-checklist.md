# 보안 체크리스트 — mimir-practice

## Mimir 보안 설정

### 인증/인가
- [ ] 멀티 테넌시 인증 활성화 (`multitenancy_enabled: true`)
- [ ] 인증 프록시 설정 (Grafana Agent, nginx 등)
- [ ] X-Scope-OrgID 헤더 검증
- [ ] 테넌트별 접근 격리 확인

### 스토리지 보안
- [ ] S3 IRSA 또는 서비스 계정 인증 (장기 자격증명 금지)
- [ ] S3 버킷 암호화 활성화 (SSE-S3 또는 SSE-KMS)
- [ ] S3 버킷 퍼블릭 접근 차단
- [ ] S3 버킷 정책으로 Mimir 서비스계정만 접근 허용

### 네트워크 보안
- [ ] Mimir 컴포넌트 포트 내부망 제한
- [ ] TLS 설정 (컴포넌트 간 통신)
- [ ] NetworkPolicy로 컴포넌트 간 접근 제한

### 데이터 보안
- [ ] 민감 메트릭 레이블 수집 금지 정책
- [ ] 테넌트 데이터 격리 검증
- [ ] 감사 로그 활성화

## 정기 보안 점검 (월별)
- [ ] 테넌트 접근 권한 리뷰
- [ ] S3 버킷 접근 로그 검토
- [ ] 미사용 테넌트 데이터 정리
- [ ] Mimir 버전 보안 업데이트 확인
